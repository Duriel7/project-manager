**Français :** Ce ReadMe sera écrit <u>en anglais et en français</u>, de manière à ce que plus de gens sachent de quoi il s'agit.

## What is it? - Qu'est-ce que c'est ?

English:<br>
This is simple project I want to set up so I can follow my personal projects.

Français :<br>
C'est un projet simple que je veux mettre en place pour suivre mes projets personnels.

## What are the main components? - Que sont les principaux composants ?

English:<br>


Français :<br>
Une seule planche à post-it dans l'idée, qui contient tous les projets en visuel simple. Cliquer sur un projet ouvre sa fenêtre isolée dans laquelle se trouve les détails dudit projet.
Les détails incluent les étapes en cartes, elles aussi cliquables et ouvrables ; des annotations ; la possibilité de choisir une ou plusieurs émotîcones en guise d'étiquette ; des étiquettes proprement textuelles pour identifier rapidement les choses.
