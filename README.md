# Project Manager

**English:** This ReadMe will be written <u>in both english and french</u>, for more people to be able to know what's going on.

**Français :** Ce ReadMe sera écrit <u>en anglais et en français</u>, de manière à ce que plus de gens sachent de quoi il s'agit.

## What is it? - Qu'est-ce que c'est ?

**English:**<br>
This is simple project I want to set up so I can follow my personal projects.

**Français :**<br>
C'est un projet simple que je veux mettre en place pour suivre mes projets personnels.

## What are the main components? - Que sont les principaux composants ?

**English:**<br>


**Français :**<br>
Une seule planche à post-it dans l'idée, qui contient tous les projets en visuel simple. Cliquer sur un projet ouvre sa fenêtre isolée dans laquelle se trouvent les détails dudit projet.<br>
Les détails incluent : les étapes importantes en cartes, elles aussi cliquables et ouvrables ; des annotations ou mémos ; la possibilité de choisir une ou plusieurs émotîcones en guise d'étiquette ; des étiquettes proprement textuelles pour identifier rapidement les choses ; un champ de priorités préconçues (mais personnalisables plus tard).<br>
Ouvrir une carte d'étape mène à un niveau de profondeur supplémentaire pour des sous-étapes : on pourra y trouver un sélecteur de date optionnel si une limite de temps est voulue ; des boutons d'avancement préconçus (mais plus tard personnalisables) qui se reflèteront sur la liste dans la carte de projet ; des pastilles colorées pour un ordre de priorité interne aux priorités d'étapes.
